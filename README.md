To-Do List App (File Storage)
A desktop-based To-Do List application built with Java Swing that provides a clean interface for managing daily tasks. This project emphasizes data persistence by saving all user tasks into a local text file.

🚀 Features
Persistent Storage: Tasks are automatically saved to tasks.txt and reloaded when the app starts.

GUI Interface: Easy-to-use graphical interface built with Java Swing.

CRUD Operations: Create, Read, Update, and Delete tasks seamlessly.

Real-time Updates: The UI reflects changes immediately upon adding or removing tasks.

🛠️ Tech Stack
Language: Java

Framework: Swing (for GUI)

Storage: Local File System (.txt)
